Lag alert SQL pattern (RESEARCH.md):
```typescript
// LagAlertService.checkAndAlert() — called by QueueTimesProcessor after each successful poll
const result = await this.db.execute(sql`
  SELECT MAX(fetched_at) AS max_fetched
  FROM wait_times_history
  WHERE ts > now() - INTERVAL '1 hour'
`);
const maxFetched = result.rows[0]?.max_fetched;
const lagMin = maxFetched
  ? (Date.now() - new Date(maxFetched).getTime()) / 60_000
  : Infinity;
if (lagMin > 30 && !this.isQuietHours()) {
  await this.slackAlerter.sendLagAlert(lagMin);
}
```

Quiet hours helper:
```typescript
private isQuietHours(): boolean {
  const hour = Number(
    new Date().toLocaleString('en-US', {
      timeZone: 'America/New_York',
      hour: 'numeric',
      hour12: false,
    })
  );
  return hour >= 2 && hour < 6;
}
```

```typescript
});
```
