      Return as ParkDto[] (add ParkDto to shared/dto if not exists: { id: string, name: string, external_id: string })

    - async getWaitTimes(parkId: string): Promise<WaitTimeDto[]>
      1. Look up park's queueTimesId from catalog
      2. Fetch all attractions for this park: db.select({ id, name }).from(attractions).where(eq(attractions.parkId, parkId))
      3. For each attraction with non-null queueTimesId: redis.get(`wait:${attraction.id}`)
      4. Parse Redis value; compute is_stale: (Date.now() - new Date(value.fetched_at).getTime()) > 5 * 60 * 1000
      5. Return WaitTimeDto[] — include attractions even if Redis miss (is_stale=true, minutes=null)

    - async getCrowdIndex(): Promise<CrowdIndexResponseDto>
      const today = new Date().toISOString().split('T')[0];
      Read 5 Redis keys: crowd_index:${today} + crowd_index:{slug}:${today} for each park
      Parse or return null if key missing
      Return CrowdIndexResponseDto shape

      return this.weatherService.getForecast(date)
