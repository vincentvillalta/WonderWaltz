       - `processEvent(payload)`: switch on payload.event.type:
         - 'INITIAL_PURCHASE' / 'NON_RENEWING_PURCHASE': extract tripId from payload (RevenueCat stores it via the mobile SDK's `attributes` — the trip_id will be set by the client when initiating purchase).
