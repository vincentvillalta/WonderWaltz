       - Inject DB_TOKEN, REDIS_CLIENT_TOKEN, and `@InjectQueue('account-purge')` queue
       - `requestDeletion(userId: string, confirmed: boolean)`:
         a. If !confirmed, throw 400 `{ error: 'confirmation_required', message: 'Send confirmed: true to confirm account deletion' }`
         b. UPDATE users SET deleted_at = NOW(), updated_at = NOW() WHERE id = $1 AND deleted_at IS NULL

         h. DELETE FROM trip_preferences WHERE trip_id IN (SELECT id FROM trips WHERE user_id = $1)
         i. DELETE FROM entitlements WHERE user_id = $1
         j. DELETE FROM iap_events WHERE user_id = $1
