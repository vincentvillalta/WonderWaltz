     - checkForUpdates(tripId:) — compares local lastSyncedAt with server, returns whether refresh needed
     - syncStatus: enum (idle, syncing, synced, failed)
     - isOfflinePackageReady: Bool computed from cached data presence
     
     Uses @ModelActor for background writes (BackgroundSyncActor) to avoid blocking main thread. The SyncCoordinator delegates heavy writes to the actor and updates its @Observable state on completion.
