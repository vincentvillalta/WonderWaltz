  1. Create WizardViewModel.swift: @Observable @MainActor class.
     Properties:
     - currentStep: WizardStep enum (dates, parks, guests, budget, lodging, mustDoRides, mealPrefs, review) -- 8 total
     - draft: WizardDraft (SwiftData model)
     - isSubmitting: Bool
     - submitError: String?
     - stepProgress: Double { Double(currentStep.rawValue) / Double(WizardStep.allCases.count - 1) }
     
     Init: Takes ModelContext. Loads existing WizardDraft or creates new one. Sets currentStep from draft.currentStep.
     
     Methods:
     - advanceStep(): Save current step to SwiftData (try? modelContext.save()), increment currentStep, update draft.currentStep and draft.updatedAt.
     - goBack(): Decrement currentStep, save.
     - canAdvance: Bool -- validates current step has required data.
     - submitTrip(authService:) async: Creates trip via API (POST /v1/trips with wizard data), then triggers plan generation (POST /trips/:id/generate-plan). On success, clears draft. On failure, sets submitError.
     
     Auto-save: modelContext.save() called on every advanceStep and goBack.
