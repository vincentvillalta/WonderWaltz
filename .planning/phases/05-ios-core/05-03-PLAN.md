  <action>
    1. Create `WonderWaltzApp.swift` — the `@main` App struct:
       - In `init()`: call `CrashReportingService.initialize(dsn:)` BEFORE any view renders (IOS-14 requirement)
       - Initialize PostHog analytics with API key from config
       - Create `DependencyContainer` with all concrete implementations via `DISetup.makeContainer()`
       - Create `ModelContainer` via `ModelContainerConfig.makeContainer()` (from WWOffline)
       - Pass `DependencyContainer` into the environment for all child views
       - Root view: `AppRouter(appState:)` inside `.modelContainer(container)`
       - DSN and PostHog API keys should come from Info.plist or a Config.swift (not hardcoded)

    2. Create `DISetup.swift` — factory that wires concrete implementations:
       ```swift
       enum DISetup {
           static func makeContainer() -> DependencyContainer {
               let keychainStore = KeychainStore()
               let authService = AuthService(keychain: keychainStore)
               let apiClient = APIClient(authService: authService)
               let analytics = PostHogAnalyticsService()
               // OfflineStore created separately since it needs ModelContainer
               return DependencyContainer(
                   apiClient: apiClient,
                   authService: authService,
                   analytics: analytics
               )
           }
       }
       ```
       This is the ONLY place where concrete types are created. Feature packages never see concrete implementations.

    3. Create `AppState.swift` — `@Observable` root state:
       - `var currentRoute: AppRoute` — enum: `.splash`, `.onboarding`, `.main`
       - `var hasCompletedOnboarding: Bool` — persisted in UserDefaults (not auth state — onboarding is local UI state)
       - `var showAuthBanner: Bool` — set when user tries action needing auth but isn't authenticated
       - `var authBannerMessage: String` — "No internet -- connect to save your trip" (per CONTEXT.md)

    4. Create `AppRouter.swift` — NavigationStack-based router:
       ```swift
       struct AppRouter: View {
           @State private var appState: AppState
           @Environment(DependencyContainer.self) private var deps

           var body: some View {
               ZStack {
                   switch appState.currentRoute {
                   case .splash:
                       SplashView(onComplete: handleSplashComplete)
                   case .onboarding:
                       OnboardingContainerView(onComplete: handleOnboardingComplete)
                   case .main:
                       MainTabView() // placeholder for now — wizard/plan added in Plans 04/05
                   }

                   if appState.showAuthBanner {
                       // Gentle banner at top per CONTEXT.md
                   }
               }
           }
       }
       ```
       - After splash: if `hasCompletedOnboarding` -> `.main`, else -> `.onboarding`
       - After onboarding: set `hasCompletedOnboarding = true`, route to `.main`
