**turbo.json** — use `tasks` key (NOT `pipeline` — that is Turborepo v1 and will deprecation-warn in v2):
```json
{
  "$schema": "https://turborepo.com/schema.json",
  "ui": "tui",
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**", ".next/**", "!.next/cache/**", "generated/**"]
    },
    "typecheck": {
      "dependsOn": ["^build"],
      "outputs": []
    },
    "lint": {
      "dependsOn": [],
      "outputs": []
    },
    "test": {
      "dependsOn": ["^build"],
      "outputs": ["coverage/**"]
    },
    "dev": {
      "cache": false,
      "persistent": true
    }
  }
}
```

CRITICAL: `apps/ios` and `apps/android` must NOT have package.json files — they are compiled by Xcode Cloud and Gradle/GitHub Actions respectively. Never include them in `turbo run`.
